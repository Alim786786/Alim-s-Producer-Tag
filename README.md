# Alim-s-Producer-Tag 🎛️

> A VS Code extension that plays a producer tag when your terminal command fails.
> Your bugs now have a soundtrack.

---

## What does it do?

You run a command in the integrated terminal.
If it exits with an error, this extension hits you with a producer tag.

Success = silence.
Failure = FAHHHHHHH.

---

## Why does this exist?

Because normal error messages are boring.
If your build is going to crash, it might as well crash with style.

---

## Features

- 🎵 Plays a producer tag on terminal errors (non-zero exit code)
- 🎵 Rotates through multiple tags so your failures stay fresh
- 🎵 Includes a test command to preview the sound

---

## Requirements

- VS Code
- Speakers (or headphones if you enjoy privacy)
- Emotional readiness for dramatic feedback

---

## FAQ

**Q: Does it play when commands succeed?**  
A: No. We respect winning quietly.

**Q: Does this fix my code?**  
A: Absolutely not. It just produces your pain.

**Q: Is this professional?**  
A: Define "professional."
